# React-Native-Redux-Todo
---

### What you will need
* react-native-cli
* Android Studio w/ appropriate settings (mentioned in React Native's documentation)
* JDK 8 or newer
* Watchman
* Physical device or Emulator

---
### Installation
```
npm instal --save
react-native run-android
npm start //unnecessary if already started
```
